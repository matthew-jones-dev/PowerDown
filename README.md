# PowerDown
PowerDown automatically shuts down your PC when Steam, Epic, or other launchers finish downloading. For gamers with slow internet: start downloads, sleep soundly, and save power. Windows, Linux, and macOS supported.

### Currently Supported Launchers:
- None

### Launcher Support in Development:
- Steam
- Epic

### Launchers to be added in future:
- EA App
- GOG Galaxy
- Battle.net
